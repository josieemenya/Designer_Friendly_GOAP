### GOAP Plugin adapted from a [project](https://github.com/josieemenya/ai-project)

The Designer Friendly GOAP plugin is exactly what the name intends to be; a designer friendly way to integrate Goal Oriented Action Planning into Unreal Engine 5.

#### Features :

- GOAP actor component

- Utility AI in the form of a Utility Reasoner Component

- Actions

- Goals

- An Example Controller, for those who just want to plug and play
